# Security Policy

## Supported versions

Only the latest version on the default branch is supported.

## Reporting a vulnerability

If you discover a security issue, please **do not** open a public issue with exploit details.
Instead, open a GitHub issue titled **"Security report"** with minimal details, or contact the maintainer privately.
